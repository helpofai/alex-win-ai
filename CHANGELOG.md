# Changelog: Alex AI Agentic Core

## [4.0.0] - 2025-12-23
### Added
- **Society of Brains Architecture**: CEO-Brain orchestrator with Plan-Execute-Reflect loop.
- **Zero-Trust Security**: Multi-factor authentication with Face ID and Voice Fingerprinting.
- **Transparency HUD**: Real-time action preview, live monitor, and critical confirmation windows.
- **App Discovery System**: Automatic indexing of Windows EXE and Microsoft Store (UWP) apps.
- **Vision Cortex**: Real-time local OCR and UI grounding for semantic screen interaction.
- **Episodic Memory**: Experience-based learning that remembers successful task sequences.
- **Self-Evolution**: Ability for the AI to write and install its own Python skills.
- **Developer Co-Pilot**: Codebase awareness and visual debugging integration.
